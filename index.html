FLex Box

We will talk about Flexbox (commonly known as Flex), which is the most common and most supported by browsers. What Flexbox does is control the structure of a group of elements that fall under one container, and it allows us to: Describe these elements on one line without specifying the width of each one (and using float). It also adds the elements to a new line if space is not enough. Arrange these items in a column with great ease. Align items right, left, or center (relative to the container). Change the order in which elements appear without modifying the HTML. Determine how much space each item takes without worrying about changing the size of the container. More details
